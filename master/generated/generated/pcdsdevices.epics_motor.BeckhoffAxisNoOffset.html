pcdsdevices.epics_motor.BeckhoffAxisNoOffset

class pcdsdevices.epics_motor.BeckhoffAxisNoOffset(*args, **kwargs)

A BeckhoffAxis with various fields read-only.

This is to prevent a user from messing themselves up by changing things like user offset and user direction. This is desirable for static beamline configurations.

Rather than removing these entirely, keep them readable in case it is useful to know their values.
