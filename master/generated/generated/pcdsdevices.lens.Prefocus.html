pcdsdevices.lens.Prefocus

class pcdsdevices.lens.Prefocus(prefix, *, name, **kwargs)

PreFocussing Lens Stack (PFLS).

Positions stack to one of three states or ‘OUT’ using a combined state record controlling an X and a Y motor.

Parameters

  • prefix (str) – The EPICS base PV of the lens stack.

  • name (str) – A name to refer to the device.
