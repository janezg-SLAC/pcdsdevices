pcdsdevices.ipm.IPIMB

class pcdsdevices.ipm.IPIMB(prefix, *, name, prefix_ioc=None, **kwargs)

Class for an IPIMB box.

Parameters

  • prefix (str) – IPIMB base PV.

  • name (str) – Alias for the IPIMB.

  • prefix_ioc (str) – IPIMB base PV for IOC PVs.

isum

Total sum of all 4 channels (i0 if standard IPM device).

xpos, ypos

X&Y beam position position calculated from 4 channels.

bias

Voltage applied to diodes in V.

delay

Delay of trigger relative to input EVR.

evr_channel

Trigger component.
