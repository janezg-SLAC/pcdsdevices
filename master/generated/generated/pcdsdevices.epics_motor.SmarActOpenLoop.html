pcdsdevices.epics_motor.SmarActOpenLoop

class pcdsdevices.epics_motor.SmarActOpenLoop(prefix='', *, name, kind=None, read_attrs=None, configuration_attrs=None, parent=None, **kwargs)

Class containing the open loop PVs used to control an un-encoded SmarAct stage.

Can be used for sub-classing, or creating a simple device without the motor record PVs.
