pcdsdevices.signal.InternalSignal

class pcdsdevices.signal.InternalSignal(*args, **kwargs)

Class Signal that stores info but should only be updated by the class.

SignalRO can be updated with _readback, but this does not process callbacks. For the signal to behave normally, we need to bypass the put override.

To put to one of these signals, simply call put with force=True
