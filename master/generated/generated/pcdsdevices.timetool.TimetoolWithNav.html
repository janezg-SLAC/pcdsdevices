pcdsdevices.timetool.TimetoolWithNav

class pcdsdevices.timetool.TimetoolWithNav(prefix, *, name, prefix_det, **kwargs)

Timetool motion with zoom, focus, and x- & y-motion motors.

The PVs for each of the motors will be inferred from the base prefix but the area detector’s PV must be passed as a keyword argument, as labeled.

Parameters

  • prefix (str) – The EPICS base PV of the Timetool.

  • name (str) – A name to refer to the device.

  • prefix_det (str) – The EPICS base PV of the detector.
