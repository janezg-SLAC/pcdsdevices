pcdsdevices.device.UnrelatedComponent

class pcdsdevices.device.UnrelatedComponent(*args, **kwargs)

Components that are unrelated to the main prefix attribute.

These components each need to have a full prefix specified rather than the normal suffix extension or formatting. This is specified in the device’s **kwargs, e.g. subdevice_prefix='SOME:PV'.

The collect_prefixes class method is used to gather these special prefixes in a convenient one-liner rather than needing to sift through formatted components.

Examples

A simple usage example:

from pcdsdevices.device import UnrelatedComponent as UCpt

class MyDevice(Device):
    motor1 = UCpt(EpicsMotor)
    motor2 = UCpt(EpicsMotor)

    def __init__(self, prefix, **kwargs):
        UCpt.collect_prefixes(self, kwargs)
        super().__init__(prefix, **kwargs)
